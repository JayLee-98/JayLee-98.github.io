Ketchup

Ketchup은 업무에 필요한 모든 기능을 하나로 모은 All-in-one 그룹웨어 시스템입니다.

Technology Stack

Spring Boot React.js Redux FullCalendar.js Google Calendar API MySQL Git Notion Figma

Key Features

  • 일정 관리: FullCalendar.js를 활용하여 각 부서별로 일정 관리가 가능합니다. 일정 추가, 수정, 삭제 등의 기능을 제공합니다.
  • 예약 관리: 회의실, 설비 등의 자원을 예약할 수 있는 기능을 구현하여 회사 자원을 효율적으로 관리할 수 있게합니다.
  • 파일 관리: 각 부서 별로 파일을 다운로드하거나 업로드할 수 있는 일원화된 파일 저장소 기능을 수행합니다.
  • 전자 결재: 온라인으로 결재를 진행할 수 있는 전자 결재 시스템을 구현했습니다. 각 결재 라인에서 승인을 받아야 다음으로 넘어가도록 개발하였습니다.

Project Outcome

프로젝트 결과

Ketchup은 다양한 업무 기능을 하나의 플랫폼으로 통합하여 회사의 효율성을 극대화했습니다. 이를 통해 사용자는 일정 관리, 예약, 파일 관리, 전자 결재 등 모든 기능을 한 곳에서 관리할 수 있게 되었습니다.
