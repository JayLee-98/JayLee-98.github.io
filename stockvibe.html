StockVibe

StockVibe는 주식 및 투자 관련 데이터를 분석하여 사용자에게 실시간으로 제공하는 플랫폼입니다. 투자 결정을 돕기 위해 다양한 지표와 데이터를 시각화하여 제공합니다.

Technology Stack

Python React.js FastAPI Google Gemma 2 Styled-components Firebase MySQL AWS Git Notion Figma

Key Features

  • 실시간 데이터 분석: 실시간으로 주식 데이터를 수집하고 분석하여 사용자에게 최신 정보를 제공합니다.
  • 데이터 시각화: 복잡한 투자 데이터를 이해하기 쉽도록 차트와 그래프 형태로 시각화하여 제공합니다.
  • 뉴스 분석: 주식 정보에 대한 뉴스를 분석하여 기사 내용을 요약하고 긍부정 감정하는 기능을 제공합니다.
  • 투자자 여론 체크: 주식 종목 토론방의 유저 토론을 종합 요약하고, 긍부정 감정을 통해 투자자들의 여론을 분석하는 제공합니다.
